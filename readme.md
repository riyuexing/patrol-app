
# 智巡煤矿 (Smart Inspection System)

## 一、 项目概述
本应用是一款专为煤矿生产现场设计的**工业级安全巡检系统**。采用“离线优先”设计理念，旨在解决井下无信号环境下的数据记录难题。应用通过高保真 UI 交互和严谨的业务闭环，确保每一项隐患都能“发现-整改-销号”。

---

## 二、 核心功能特性

### 1. 地点库与硬件资产管理 (Enhanced)
为了防止巡检人员“空巡”并确保数据的真实性，系统建立了完备的点位档案库与物理硬件映射机制：
- **档案维护 (CRUD)**：支持点位名称、区域、编码的标准化管理。
- **批量导入导出 (New)**：支持通过 CSV 文件一键导入巡检点档案。可在 Excel 中预设点位名称、编码、NFC UID 及 QR ID，实现硬件资产的批量数字化录入。
- **批量硬件管理 (New)**：在管理界面支持“多选模式”，可对选中的多个点位执行批量解绑、批量绑定虚拟 NFC 标签或资产二维码。
- **NFC 硬件生命周期**：
    - **唯一性绑定**：读取 NFC 标签的全球唯一 UID 并将其与数字点位锁定。
    - **无损更换**：支持“更换标签”流程，在保留历史巡检轨迹的前提下建立新 UID 映射。
- **二维码双模式管理**：
    - **系统生成码**：由系统根据点位 ID 自动生成，支持无限次重印。
    - **外部资产码**：支持扫描已有的设备名牌（资产贴纸），实现“一码一地”的快速关联。

### 2. 多维度感应巡检
- **NFC 触碰巡检**：利用手机 NFC 传感器，靠近标签即刻调起对应地点的巡检工单。
- **二维码扫描巡检**：识别系统码或已绑定的外部资产码。

### 3. 极速巡检与整改闭环
- **快速模式**：支持“拍照+状态+备注”极速完成。新增“现场即时整改”功能。
- **整改追踪时间轴**：清晰记录“发现异常 -> 多次整改追加 -> 销号确认”的完整生命周期。

### 4. 语音集成
- 集成 Web Speech API，支持在备注、整改说明等关键输入框使用语音转文字。
