# 智巡煤矿 App 优化与功能路线图 (V1.2.0+)

作为产品经理，基于煤矿井下“高风险、弱光线、重合规、离线环境”的实际应用场景，以下是后续版本的优化方向及功能优先级排序。

---

## 🚀 优先级排序 (Priority List)

### P0：合规性与真实性 (核心红线)
- [x] **手写电子签名 (E-Sign)**: 满足“谁检查、谁签字、谁负责”的法律合规要求。
- [ ] **GPS/北斗/基站位置校验**: 记录巡检时的坐标，防止人员在非现场“隔空打卡”。
- [ ] **离线数据加密存储**: 针对工业手持终端，确保本地数据库文件在设备丢失时不被非法提取。

### P1：智能化与减负 (AI 辅助)
- [x] **Gemini AI 视觉隐患识别**: 自动分析巡检照片，识别皮带跑偏、杂物堆放、支架漏液等常见隐患。
- [ ] **语音自动化工单生成**: 巡检员边走边说，AI 自动提取“地点”、“隐患类型”、“级别”并自动填表。
- [ ] **低光照图像增强**: 针对矿下弱光环境，利用模型对现场取证照片进行增强处理，确保回传清晰。

### P2：任务驱动与看板 (流程效率)
- [ ] **巡检任务日历/看板**: 按照日、周、月展示待检任务，变“主动找活”为“任务驱动”。
- [ ] **临期/过期未检提醒**: 对重要瓦斯点位、配电房等高频点位进行强提醒。
- [ ] **班组对比排行**: 展示各班组的巡检覆盖率与隐患闭环速度，引入激励机制。

### P3：协作增强 (精细化管理)
- [ ] **图片红圈标注 (Annotation)**: 巡检员在隐患照片上直接涂鸦标注，让整改人员一眼看清问题点。
- [ ] **一键导出标准化 Word/PDF 台账**: 自动生成符合国家煤监局要求的 A4 标准化纸质归档资料。

---

## 🛠 详细功能说明 (Scenarios)

### 1. 轨迹与位置校验 (P0)
*   **场景**: 井下巡检有时存在“抄近路”或在休息室“虚假扫码”的情况。
*   **行为**: 当触发 NFC/扫码时，系统后台记录当前的经纬度或近距离基站 ID。如果坐标偏差过大，系统发出弹窗警告并要求拍摄一张带有环境参照物的全景照片。

### 2. 图片标注功能 (P3)
*   **场景**: 在复杂的机电设备照片中，微小的裂纹或渗油很难在小屏幕上识别。
*   **行为**: 用户在查看详情或提交整改时，点击图片进入编辑模式，可选择“红色圆圈”或“箭头”指向问题区域。

### 3. 智能任务日历 (P2)
*   **场景**: 巡检员班次多，容易遗漏非每日检点位（如周检点）。
*   **行为**: 首页增加一个日历视图，绿色点代表已闭环，红色点代表有遗留隐患，灰色代表今日待检。点击日期即可快速开始当天的工单。
